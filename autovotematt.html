Auto-Vote Matt on topstreamers.com

Via cron on Unix (Linux and OS X)

cron is a job scheduler on Unix system that allows to run tasks periodically. First of all, download this little script and put it in a new folder in your home directory named Scripts (you can create this folder by typing mkdir ~/Scripts in a Terminal).

Then all we have to do is to create the task for cron. To do that, open a Terminal and type crontab -e. Now you're editing the configuration of cron in vi (a powerful text-editor made for Terminal. To start editing the file, you need to press i. You're now in insert mode (you see -- INSERT -- in the lower-left corner of the Terminal)

Now you have to write the task you want (which is launching the file ~/Scripts/voteMatt.sh). So you can learn how to program cron task if you want on the wikipedia page but here's the one I did: 0 */2 * * * ~/Scripts/voteMatt.sh 2>&1. This task will run the script every 2 hours at 0 minute (so at midnight, 2AM, 4AM, etc.). It will only count one vote a day obviously but at least you're almost sure that that you will vote each day (also, if in a day you have several IP, it will count another vote). Also, I added 2>&1 to the end of the command, so cron won't email me in case of error (and the error will be stored into the regular output, here it's in vote.log)

To finish the editing, you have to press Esc and type :wq (to write the file and quit the editor, and do not forget the :). After that, cron should say crontab: installing new crontab. You can check that your task is in place by executing crontab -l.

And now your computer will vote every day!

Via OS X Calendar

I've made a little script (basically just a little curl command) wrapped into an App. You can download it here and if you're curious here is the little command: curl 'http://mattmelvin.topstreamers.com/ajax/vote' --data 'id=4073'

So now if you run that little App, it will vote for Matt on Top Streamer (it won't show anything though, it closes automatically). If you can't launch it, you may have to allow to launch Apps from all developers (go in System Preferences > Security & Privacy > Tab General, click the lock in the lower-left corner, and then allow to run Apps from all developers)

Now, in order to run automatically this App, I used Calendar. Launch Calendar, go in Preferences (Cmd + ,), in Accounts tab disable every account you have (you'll re-enable them later). Close Preferences and create a new calendar (File > New Calendar > On My Mac) and call it "Scripts". Now create a new all-day event on today, repeat it daily, name it "Vote Matt" and set an alarm "Otherâ€¦" and select "Launch a file" (the file to select is obviously the App you downloaded).

You can finally re-enable all your calendar accounts, also in the Alerts tab of Calendar preferences you have to remove all alerts from the account "On My Mac". Then on the calendar view you can uncheck the "Script" calendar to not show all the events.

Now your Mac will automatically vote every day!
